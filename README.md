# Info-Tech-org 默认社区文件与组织展示

本仓库是 [Info-Tech-org](https://github.com/Info-Tech-org) 组织的特殊配置仓库：

- **组织首页展示**：[profile/README.md](profile/README.md) 会显示在 [github.com/Info-Tech-org](https://github.com/Info-Tech-org) 的概览页
- **默认社区健康文件**：本仓库根目录下的 `CODE_OF_CONDUCT.md`、`CONTRIBUTING.md`、`SECURITY.md` 以及 `ISSUE_TEMPLATE/`、`PULL_REQUEST_TEMPLATE.md` 会作为组织内未单独配置的仓库的默认模板
- **组织原则**：[ORG_PRINCIPLES.md](ORG_PRINCIPLES.md) 阐明使命、愿景与协作原则

详见 [GitHub 文档：创建默认社区健康文件](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file)、[自定义组织资料](https://docs.github.com/en/organizations/collaborating-with-groups-of-people/customizing-your-organizations-profile)。
